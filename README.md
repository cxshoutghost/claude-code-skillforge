## 📄 About claude-code-skillforge

Claude Code Skillforge is a desktop application designed to help you create, improve, and review skill files. It works with Claude Code and Antigravity SKILL.md files. The tool keeps your skills up-to-date with best practices, lets you search a marketplace for skill ideas, and checks your work for errors automatically. It works on Windows and runs without needing technical skills.

This app focuses on making skill management simple and fast, with features like live syncing, skill upgrades, scanning, and self-validation. You control your files and get fast results in minutes.

## ⚙️ Basic Usage

1. Open claude-code-skillforge from your desktop or Start menu.

2. Use the **File** menu to create a new skill or open an existing SKILL.md file.

3. The main window shows your skill with options to build, upgrade, scan, or validate.

4. To upgrade your skill:
   - Click the **Upgrade** button
   - The app will update your file using the latest guidelines

5. To scan for errors:
   - Select the **Scan** option
   - The app highlights issues and suggests fixes

6. To search the skill marketplace:

   - Type keywords to find similar skill files or templates

7. Save your changes regularly by clicking **Save**.
